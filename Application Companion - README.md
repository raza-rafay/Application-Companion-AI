# **Application Companion AI**

Application Companion AI is a Streamlit based tool that helps students and early career professionals evaluate their fit for internships and roles by comparing a resume against a job description.

The tool provides:

* A resume to job fit overview

* Matched and missing skills

* Keyword analysis

* Clear indicators of alignment gaps

## **Motivation**

Applying to internships often involves manually tailoring resumes for each role. This project was built to better understand resume screening logic and to create a structured way to evaluate resume fit while applying.

The idea emerged during internship applications, with the goal of improving clarity, focus, and decision making before submitting applications.

## **Features**

* Resume and job description comparison

* Keyword extraction and matching

* Job coverage scoring

* Resume relevance scoring

* Identification of missing and extra skills

## **Tech Stack**

* Python

* Streamlit

* Regular expressions

* Basic natural language processing techniques
